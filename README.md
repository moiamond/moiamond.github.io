## hexo

本部落格是由[hoxo](https://hexo.io)產生。

### 安裝 hexo

使用npm安裝hexo

```bash
$ npm install -g hexo-cli
```
> -g表示安裝到系統目錄

安裝完後，執行以下命令初始化

```
$ hexo init <blog folder>
$ cd <blog folder>
$ npm install
```

Reference:
* [Getting Started | Overview](https://hexo.io/zh-tw/docs/index.html)
* [Getting Started | Setup](https://hexo.io/zh-tw/docs/setup.html)
