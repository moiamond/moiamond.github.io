使用 Docker 快速架設 RTMP 串流平台

By moiamond

發表於 2016-07-04

文章目錄

為了在開發機器上測試RTMP串流，研究了一下目前可行的方案，最後選定用Nginx+RTMP module+Docker來方便的達到此需求。

過程大概就是下面四個步驟:

  1. 安裝所有能編譯Nginx的相關套件
  2. 到Nginx的官網抓取最新的原始碼
  3. Git clone 最新的 nginx-rtmp-module
  4. 開始編譯nginx原始碼 (要打開打開nginx-rtmp-module編譯選項)

為了之後可以很方便的無腦啟動，因此整理成了Dockerfile，詳細的步驟可以在此找到
